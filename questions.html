Question-1: What are the different ways to select an element in the DOM?

1. getElementById('id') – Selects an element by its unique ID.
2. querySelector('selector') & querySelectorAll('selector') – Selects the first or all matching elements using CSS selectors.
3. getElementsByClassName('class'), getElementsByTagName('tag'), & getElementsByName('name') – Select elements by class, tag, or name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML – Returns or sets the HTML content inside an element, including tags (bold will be rendered as bold).
2. innerText – Returns or sets only the visible text inside an element, ignoring hidden elements and maintaining formatting.
3. textContent – Returns or sets all text inside an element, including hidden text, but treats everything as plain text (ignores HTML tags).

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements, using event bubbling. This improves performance by attaching a single event listener to the parent instead of multiple listeners on individual child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM mechanism where an event triggered on a child element propagates up through its ancestor elements. It starts from the target element and moves up to the root (document), allowing parent elements to respond to events fired on their children.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create an element – Use document.createElement('tag'), e.g., const div = document.createElement('div');.
2. Add an element – Use parent.appendChild(child) or parent.append(child), e.g., document.body.appendChild(div);.
3. Remove an element – Use element.remove() or parent.removeChild(child), e.g., div.remove();.
